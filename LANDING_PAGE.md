# Landing Page do FisioTech

## 🎉 Nova Landing Page Criada!

Uma landing page completa e profissional foi criada para apresentar o produto FisioTech.

### ✨ Características da Landing Page:

- **Design Moderno**: Gradientes, cards com hover effects, animações suaves
- **Seções Completas**:
  - Hero section com CTAs principais
  - Recursos principais do sistema
  - Planos e preços (Starter R$97, Professional R$197, Enterprise R$397)
  - Depoimentos de clientes
  - Call-to-action final
  - Footer completo

- **Interatividade**: 
  - Todos os botões têm hover effects
  - Animações de hover em cards
  - Efeitos de escala nos botões
  - Transições suaves

- **Funcionalidade**:
  - ✅ Botão "Entrar" → redireciona para `/auth`
  - ✅ Botão "Cadastrar" → redireciona para `/auth`
  - ✅ Botão "Começar Grátis" → redireciona para `/auth`

### 🚀 Como Acessar:

1. **Desenvolvimento**: Acesse `http://localhost:5173/landing`
2. **Produção**: Acesse `https://seudominio.com/landing`

### 📁 Arquivo Criado:

- `src/components/LandingPage.tsx` - Componente principal da landing page
- Rota adicionada em `src/App.tsx` como rota pública

### 🎨 Tecnologias Utilizadas:

- React + TypeScript
- Tailwind CSS
- Shadcn/ui components
- Lucide React icons
- React Router para navegação

### 📱 Responsivo:

- Layout adaptativo para desktop, tablet e mobile
- Cards reorganizados automaticamente
- Texto e botões otimizados para diferentes tamanhos de tela

A landing page está pronta para uso e pode ser customizada conforme necessário!
